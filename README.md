# 🌱 Carbonest – AI-Powered Carbon Footprint & Financial Advisory System

## 📌 Overview

Carbonest is a full-stack intelligent platform designed to help users track, analyze, and optimize their carbon footprint while integrating financial insights. The system combines machine learning, backend APIs, and an interactive frontend dashboard to provide actionable recommendations.

## 🧠 Key Features

* 📊 Carbon footprint tracking and analysis
* 🤖 AI-based recommendations
* 📈 Financial advisory integration
* 🔗 REST API with Swagger documentation
* 🖥️ Interactive frontend dashboard
* 🔐 User authentication and management

---

## 🏗️ System Architecture

* Frontend: Web-based UI (Netlify deployment)
* Backend: REST API (Spring Boot / deployed on Render)
* Data Flow: User → Frontend → Backend API → Processing → Response

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Spring Boot (Java)
* **API Docs:** Swagger/OpenAPI
* **Deployment:**

  * Frontend → Netlify
  * Backend → Render
